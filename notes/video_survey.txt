```
+--------------------+
| Caméra 1 (RTSP)    |
```

## 📡 5. Diffusion en direct multi-caméras (via FFmpeg + HLS)

### Principe :

Chaque caméra RTSP → convertie en flux HLS (HTTP Live Streaming) via FFmpeg.
Chaque flux produit un `.m3u8` unique.

### Exemple de script Bash (multi-caméras) :

Crée un fichier `scripts/start_streams.sh` :

```bash
#!/bin/bash

mkdir -p public/streams

# Exemple : 3 caméras
ffmpeg -i rtsp://user:pass@192.168.1.10/stream -f hls -hls_time 2 -hls_list_size 3 -hls_flags delete_segments public/streams/camera1.m3u8 &
ffmpeg -i rtsp://user:pass@192.168.1.11/stream -f hls -hls_time 2 -hls_list_size 3 -hls_flags delete_segments public/streams/camera2.m3u8 &
ffmpeg -i rtsp://user:pass@192.168.1.12/stream -f hls -hls_time 2 -hls_list_size 3 -hls_flags delete_segments public/streams/camera3.m3u8 &
```

Lance :

```bash
bash scripts/start_streams.sh
```

➡️ Chaque caméra aura son flux dans `/public/streams/cameraX.m3u8`.

## 💾 7. Enregistrement automatique en base de données

Crée un modèle :

```bash
php artisan make:model Video -m
```

Migration :

```php
Schema::create('videos', function (Blueprint $table) {
    $table->id();
    $table->foreignId('camera_id')->constrained()->onDelete('cascade');
    $table->string('filename');
    $table->binary('file_data');
    $table->unsignedBigInteger('size')->nullable();
    $table->timestamps();
});
```

---

### Commande Laravel d’enregistrement (multi-caméras)

```bash
php artisan make:command RecordCameras
```

`app/Console/Commands/RecordCameras.php` :

```php
namespace App\Console\Commands;

use Illuminate\Console\Command;
use App\Models\Camera;
use App\Models\Video;

class RecordCameras extends Command
{
    protected $signature = 'record:cameras';
    protected $description = 'Enregistre automatiquement les vidéos des caméras en base';

    public function handle()
    {
        foreach (Camera::all() as $camera) {
            $filename = 'camera_' . $camera->id . '_' . now()->format('Ymd_His') . '.mp4';
            $tempPath = storage_path('app/temp/' . $filename);

            if (!file_exists(dirname($tempPath))) mkdir(dirname($tempPath), 0777, true);

            // Enregistre 30 secondes du flux
            $cmd = "ffmpeg -y -i {$camera->stream_url} -t 30 -vcodec copy -acodec copy $tempPath";
            exec($cmd);

            if (file_exists($tempPath)) {
                $data = file_get_contents($tempPath);
                $size = filesize($tempPath);

                Video::create([
                    'camera_id' => $camera->id,
                    'filename' => $filename,
                    'file_data' => $data,
                    'size' => $size,
                ]);

                unlink($tempPath);

                $this->info("Vidéo enregistrée : $filename");
            }
        }
    }
}
```

Exécute :

```bash
php artisan record:cameras
```

## 🔁 8. Planifier les enregistrements automatiques

Dans `app/Console/Kernel.php` :

```php
protected function schedule(Schedule $schedule)
{
    $schedule->command('record:cameras')->everyTenMinutes();
}
```

Puis lance :

```bash
php artisan schedule:work
```

```php
use App\Models\Video;
use Illuminate\Support\Facades\Response;

Route::get('/videos/{video}', function (Video $video) {
    abort_unless(Auth::check(), 403);

    return Response::make($video->file_data, 200, [
        'Content-Type' => 'video/mp4',
        'Content-Length' => $video->size,
        'Content-Disposition' => 'inline; filename="'.$video->filename.'"'
    ]);
})->name('videos.show');
```
